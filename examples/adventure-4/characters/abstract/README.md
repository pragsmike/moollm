### âš ï¸ The Prototype Paradox

The `player/` entity is a **prototype** â€” a template from which other characters can be cloned. You have two choices:

1. **Clone it** (recommended): Create a new character based on the template

**Why is playing the prototype risky?**

> *"I accidentally played as the prototype for several turns before I realized. I quickly instantiated a proper character and switched to that â€” imagine if I'd made Player permanently grumpy, or dressed them in a clown suit that all future characters inherited!"*

